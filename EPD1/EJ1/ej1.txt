PROYECTO WEB APLICATION
Un proyecto con  web Application está orientado a aplicaciones en Java las cuales usa servlets, JSPs y frameworks.
Estas, necesitan un servidor de aplicaciones como GlassFish o  Tomcat y son mas complejas ya que estan pensadas
para proyectos grandes con lógica de servidor.

PROYECTO PHP APPLICATION
Un proyecto con php Application está orientado al lenguaje PHP que se suele usar en el backend, el cual permite conectar con bases de datos MySQL/MariaDB fácilmente.
Es ideal para desarrollar sistemas web mas dinámicos.

PROYECTO HTML5/JS APLICATION
Un proyecto con HTML5/js Application está centrado en el frontend, el cual incluye soporte para frameworks JS como Angular o React.
No depende de servidor excepto que tú lo configures y es perfecto para sitios estáticos o SPAs, los cuales se envian directamente al navegador del usuario sin necesidad de procesar por el servidor.
